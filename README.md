# Card Recognition Engine

Scaffolded repository for the roadmap in `roadmap.md`.

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[dev]
pytest
```

## Current status

This is an initial skeleton with:

- core API and datamodels
- placeholder detector/normalizer/OCR/matcher/scorer pipeline
- catalog, adapter, and UI module placeholders
- scripts and baseline tests
